## 📂 Included Scripts & Tools

This repository contains various sub-projects. Here is an overview of what each tool does:

### 🌐 Network & Spoofing Tools
* **`Arp-Spoofing-main`**: A script that falsifies ARP messages over a local area network to intercept or manipulate traffic (Man-in-the-Middle).
* **`dns_spoofer-main`**: Intercepts DNS requests and redirects the target's web traffic to a specified IP address.
* **`mac_changer-main`**: Easily spoof or change the Media Access Control (MAC) address of your network interface for anonymity.
* **`network_scanner-main`**: Scans the local network to discover connected devices, displaying their IP and MAC addresses.
* **`packet_sniffer-main`**: Captures and analyzes network packets in real-time. Useful for extracting unencrypted data like HTTP URLs and plaintext passwords.

### 🕷️ Web Crawling & Scanning
* **`Web-Crawler-Link-Extractor-main`**: Crawls target websites to extract and list all internal and external links.
* **`directory_crawler-main`**: Discovers hidden directories and paths on a web server using wordlist-based enumeration.
* **`subdomain_crawler-main`**: Identifies hidden or unlinked subdomains of a target website to expand the attack surface area.
* **`vulnerability_scanner-main`**: Scans web applications for common security flaws (like XSS or misconfigurations) and generates a report.

### 🔐 System & Exploitation Tools
* **`Backdoor_python-main`**: A basic Python-based reverse shell/backdoor allowing remote code execution and system access.
* **`Brute_force-main`**: A tool designed to brute-force login pages or services using custom wordlists.
* **`Key_logger-main`**: Runs in the background to record keystrokes, useful for understanding how input monitoring malware operates.
